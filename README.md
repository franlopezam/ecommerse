# 🛒 E-Commerce Web

Una plataforma de **comercio electrónico moderna** desarrollada para vender productos en línea de manera rápida, segura y fácil de usar.

Este proyecto permite a los usuarios **explorar productos, agregarlos al carrito y realizar compras**, con una interfaz optimizada para **computadoras y dispositivos móviles**.

---

## ✨ Características

* 🛍️ Catálogo de productos
* 🔎 Búsqueda de productos
* 🛒 Sistema de carrito de compras
* 💳 Simulación de pago
* 📱 Diseño responsive (funciona en PC y celular)
* ⚡ Interfaz rápida y moderna
* 💾 Guardado de datos usando LocalStorage
* ⭐ Sistema de puntuaciones o favoritos (opcional)
